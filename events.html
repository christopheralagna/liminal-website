EVENTS

Liminal’s events will be comprised of:
on-going courses // one-time workshops // panels // community meetings // mutual aid meetups // skill-sharing // solidarity action groups // and community celebration.

All events can be understood to be diversified forms of anti-capitalist education:
Liminal believes that education should be immediately applicable, supportive, enticing, affordable, resist gate-keeping, resist competitive individualism and should simply feel empowering!
